# 푸꾸옥 여행 플랜

> 베트남 푸꾸옥 여행을 위한 올인원 PWA 앱  
> 일정 관리, 지도, 다이어리, 예산까지 한 곳에서

---

## 주요 기능

### 🗺️ 지도
- Leaflet 기반 인터랙티브 지도
- 카테고리별 장소 필터 (식당, 카페, 마사지, 쇼핑, 관광, 숙소)
- **MY루트** — 방문할 장소를 핀으로 관리
- 현재 위치 표시 (GPS)
- 장소별 메모, 가격 정보 기록
- 커스텀 장소 추가 (Google Maps 연동)

### 🗓️ 일정
- 날짜별 일정표 (숙소, 식사, 액티비티 등 9개 섹션)
- 날짜별 메모 기능
- 날씨 정보 (기온, 강수확률)
- 일정 PDF/프린트 내보내기

### 📓 다이어리
- 사진 업로드 및 캡션 작성
- 닉네임 기반 댓글/답글 (비밀번호 보호)
- 날짜별 자동 분류
- HTML/PDF 내보내기

### 💰 예산 & 팁
- **예산** — 날짜별 지출 내역 (VND ↔ KRW 환산)
- **팁** — 체크리스트, 핵심 정보 카드, 커스텀 정보 카드
- 결제수단 추적 (카드/현금)

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Vanilla JS, HTML5, CSS3 |
| 실시간 DB | Firebase Realtime Database |
| 지도 | Leaflet.js + OpenStreetMap |
| 이미지 처리 | html2canvas |
| 오프라인 | Service Worker (Cache-first) |

## 설치 방법

### GitHub Pages (추천)
1. 이 레포지토리를 Fork 또는 Clone
2. **Settings → Pages → Source**: `main` 브랜치 선택 → Save
3. `https://<username>.github.io/phuquoc/` 에서 접속
4. 폰 브라우저에서 열고 **"홈 화면에 추가"** 탭

### Netlify
1. [app.netlify.com](https://app.netlify.com) 접속 → 로그인
2. 이 폴더를 드래그 앤 드롭으로 업로드
3. 생성된 URL로 폰에서 접속 → **"홈 화면에 추가"**

### 로컬에서 바로 보기
- `index.html`을 폰으로 전송 (카톡, 에어드롭 등) 후 브라우저에서 열기
- ※ 오프라인 캐싱은 안 되지만 내용 확인 가능

## PWA 특징
- 홈 화면에 앱 아이콘으로 추가 → 네이티브 앱처럼 실행
- 오프라인 지원 (비행기 모드, 데이터 없을 때)
- 주소창 없는 풀스크린 모드
- Firebase 실시간 동기화 (일행과 데이터 공유)

---

## 관리자 모드
비밀번호 입력으로 관리자 모드 활성화:
- 일정 섹션/라벨 편집
- 장소 추가/삭제/숨기기
- 체크리스트 및 팁 카드 관리
- 다이어리 항목 삭제 및 휴지통 관리
- 탭 이름 변경, 순서 변경, 숨기기
